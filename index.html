網路線製作


網路線製作有兩種接頭，分別是
T568A
綠白 綠 橘白 藍 藍白 橘 棕白 棕
T568B
橘白 橘 綠白 藍 藍白 綠 棕白 棕
當線兩端為同接頭時，為網路線，反之則為跳線，用來使兩台電腦連接傳輸。
製作方式:
剝皮->內線(不剝皮)依上述兩方法排列->確定線頭整齊後插入接頭->確定插至最深->使用壓軸夾緊接頭

fossil 除錯

/etc/stunnel建立一個資料夾：
mkdir repository
cd repository
fossil init wcm6.fossil...........................建立wcm6.fossil
sudo /etc/init.d/stunnel4 restart.............重啟即可
